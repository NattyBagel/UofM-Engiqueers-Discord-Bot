# Discord Bot
This discord bot is created with the express purpose of lasting for as many years as possible, serving the (*not yet created*) UofM Engiqueers Discord Server. As people join and leave the club, having the functionality and foresight for the future is a must for a tool.

## Private & Personal Data
Certain commands are equipped with presenting data which might be sensitive to certain users. Presenting this data in a hidden format is important, incase this information doesn't want to be shared to others.

**No data is collected or stored when using a command**

Once a command is used, only the user is aware of it.
